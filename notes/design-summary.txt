  Things I had to resolve (flagged in §3, the architect's job, not silent 
  guesses):
  - Junction/net derivation (OQ-007): a junction endpoint stores the target wire
  id (+ a render-only coord), and nets are computed by union-find over ids only
  — never pixel geometry — satisfying FR-059a. §6.6 has the algorithm.
  - Fan-out vs two-endpoint wires: fan-out = multiple wires sharing a pin +
  junctions; no wire ever has >2 endpoints.
  - Delete-leaves-dangling-junction gap (G2): deleting a wire converts any
  junction referencing it into a free (dangling) endpoint, then the
  zero-endpoint sweep runs.
  - Grid/zoom defaults (OQ-004): 8 px/grid-unit default, 0.25×–4.0× zoom,
  rotation done about the instance origin so 90° turns keep pins on integer
  grid.
  - NFR-005 made testable: ≤16 ms frames at a stated stress size.
  - Bus-group width (A3): width = Σ member pin bit-widths, first-declared on
  ties.
